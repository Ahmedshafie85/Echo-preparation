AI backend

Claude → Groq

The app uses Claude for all clinical content — it is held to a verified set of ASE/EACVI reference values on every call. Groq is an optional fast fallback. Keys are saved in this browser only — never in the GitHub repo. Each colleague enters their own key on their own device.

⚠︎ Never paste your key into index.html or commit it — GitHub's secret scanner will detect it and Groq will auto-revoke the key. If your key stopped working, that is almost certainly why: generate a fresh one in the Groq console and enter it here.
